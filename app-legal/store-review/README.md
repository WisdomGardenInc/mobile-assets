# Store-review privacy snapshots (国内应用市场审核)

Frozen, self-contained privacy-policy pages submitted to domestic (China)
app-market review. Uploaded to Qiniu **manually** (not via the deploy scripts)
and not loaded by the app. Live URL: `https://mobile-download.tronclass.com.cn/<file>`.

**To add one:** put the new dated HTML here and upload it to Qiniu under the same
filename. Only recent official versions are mirrored here; older ones stay on Qiniu.
